# Chip8 Emulator - Haskell

### Installation and usage
Install SDL2 and Stack

In the root folder
```
stack build
stack exec chip8t-exe
```

### Controls
Chip 8 uses a keypad style controller

-------------
1	2	3	C
-------------
4	5	6	D
-------------
7	8	9	E
-------------
A	0	B	F
-------------


Most games can be started by pressing 5
